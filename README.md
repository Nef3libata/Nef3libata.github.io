## 由比ヶ浜結衣は人類の希望である
CSS Animationを使って、由比ヶ浜結衣の可愛さを僕なりに目一杯表現しました。

この場を使って、由比ヶ浜結衣というキャラクターを生み出し、命を吹き込んでくださった。

渡航先生・東山奈央さん・俺ガイル制作に関わった全ての方々に感謝します。
